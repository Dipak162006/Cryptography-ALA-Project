📚 Educational Concepts: MAC vs Signature

What is MAC / HMAC?

Message Authentication Code (MAC) guarantees that a message came from the stated sender (authenticity) and wasn't changed (integrity) via a shared secret key.

Difference: Dig-Sig vs MAC

Digital Signatures (ALA-1) use Asymmetric keys (Public/Private pair) for non-repudiation. MAC uses Symmetric cryptography (Shared Secret) meaning the sender and receiver possess the exact same key.
